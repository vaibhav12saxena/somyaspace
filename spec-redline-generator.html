The Problem

Design QA and dev handoff were painfully slow

For every new project or UI screen, designers spent ~45 minutes manually annotating spacing, padding, border radii, font sizes, and colour values before handing off to developers. This created a bottleneck — especially with new features shipping frequently.

The Solution

One click to annotate everything

I built a Figma plugin that analyses any selected frame and generates a full spec sheet — spacing redlines, colour swatches, border radii, auto-layout properties, and typography details — all placed cleanly alongside the original design.

How It Works

Select, run, done

Step 01
Select Frame
Pick any frame or component in Figma. The plugin reads its full layer tree — nested components, auto-layouts, and all.
Step 02
Analyse & Extract
The plugin extracts spacing between elements, padding, border radii, fill colours, stroke colours, font properties, and auto-layout configurations.
Step 03
Generate Annotations
Redline arrows, colour swatches, radius badges, and auto-layout info boxes are placed alongside the design — ready for developer review.
What gets annotated: Spacing between elements (horizontal & vertical), padding, border radii, fill & stroke colours with hex values, auto-layout direction, gap, padding, and alignment.
